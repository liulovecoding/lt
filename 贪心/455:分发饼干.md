**思路(也是贪心策略)**：给剩余孩子里最小饥饿度的孩子分配最小的能饱腹的饼干

**具体实现**：因为我们需要获得大小关系，所以对孩子和饼干分别排序(从小到大)
```
class Solution {
public:
    int findContentChildren(vector<int>& g, vector<int>& s) {
    sort(g.begin(),g.end());
    sort(s.begin(),s.end());
    int i=0,j=0;
    while(i<g.size()&&j<s.size()){
        if(g[i]<=s[j])
        i++;
        j++;//无论是否能喂饱都需要加一
    }
    return i;
 }    
    
};
```
